1 GitBash commands

1) Посмотреть где я - pwd
2) Создать папку - mkdir QA30
3) Зайти в папку - cd QA30
4) Создать 3 папки - mkdir QA30_1 QA30_2 QA30_3
5) Зайти в любую папку cd QA30_1
6) Создать 5 файлов (3 txt, 2 json) - touch 301.txt 302.txt 303.txt 304.json 305.json 
7) Создать 3 папки - mkdir 1QA 2QA 3QA
8. Вывести список содержимого папки - ls -la
9) + Открыть любой txt файл  - cat 301.txt
10) + написать туда что-нибудь, любой текст - cat > 301.txt
11) + сохранить и выйти - ^C
12) Выйти из папки на уровень выше - cd ..

13) переместить любые 2 файла, которые вы создали, в любую другую папку. - 
mv 301.txt 302.txt QA30_2 
14) скопировать любые 2 файла, которые вы создали, в любую другую папку. - 
 cp 301.txt 302.txt QA30_3
15) Найти файл по имени  - find . -name Lena.txt 
16) просмотреть содержимое в реальном времени  - tail Lena.txt
17) вывести несколько первых строк из текстового файла - head -n 4 Lena.txt
18) вывести несколько последних строк из текстового файла - tail -n 3 Lena.txt
19) просмотреть содержимое длинного файла (команда less) less +f Lena.txt (Ctrl + c, чтобы прервать и q, чтобы выйти из представления.) 
20) вывести дату и время - date
